ソウルドアウト株式会社 従業員向けFAQ

休暇有給休暇付与のタイミングを知りたい▼
正社員・契約社員は入社時に初回付与。アルバイトは勤続6カ月後に初回付与。その後は労働基準法に基づき付与されます。詳細は就業規則をご確認ください。
休暇有給休暇の取得義務について知りたい▼
10日以上付与された場合、付与日から1年以内に5日以上の取得が義務です。中途入社の正社員は入社月により必要取得日数が異なります。

5月入社→5日 / 6月→4.5日 / 7月→4日 / 8月→3.5日 / 9月→3日 / 10月→2.5日 / 11月→2.5日 / 12月→2日 / 1月→1.5日 / 2月→1日 / 3月→0.5日
休暇夏季休暇・夏休みの取得方法▼
夏季休暇制度はなく、その分有給休暇を多めに付与しています。上長とご相談の上、有給を活用してください。
休暇休職・時短について知りたい▼
状況に応じた個別案内となるため、Slackワークフローの「非公開問い合わせ」からご相談ください。
休暇代休申請はできますか▼
可能です。ルール・申請方法はMIKKE（01_AKASHI：各種マニュアル）をご確認ください。
休暇慶弔休暇の申請方法を教えてください▼
弔事休暇の対象と日数：
・配偶者・子・父母が死亡した場合 → 5日以内
・配偶者の父母・兄弟姉妹・祖父母（喪主）→ 5日以内
・配偶者の父母・兄弟姉妹・祖父母（喪主以外）→ 3日以内

AKASHI申請方法はMIKKEで確認。上長への連絡も必須です。
休暇時短の場合の半休の時間を知りたい▼
半休は1日の就業時間の中間となります。
例1：就業時間9:30〜17:00の場合 → 午前：9:30〜13:15、午後：14:15〜18:00
例2：就業時間9:00〜17:00の場合 → 午前：9:00〜13:00、午後：14:00〜17:00
勤怠・給与休日出勤の割増手当について知りたい▼
・固定残業時間以内の残業：割増25%分のみ支給
・固定残業時間を超える残業（固定残業代なしの場合も含む）：125%支給
勤怠・給与休日出勤の振替休日について知りたい▼
・所定労働時間（9:30〜18:00）未満 → 休日手当で賃金清算（割増賃金はメンバーのみ）
・所定労働時間以上 → 振替休日を取得（賃金清算なし）
※管理職は休日割増手当なし。打刻は必要です。
勤怠・給与給与明細の見方を知りたい▼
給与明細はカオナビのプロファイルブックに掲載。
給与＝当月固定給＋前月勤怠実績（残業代等）−住民税・社会保険料等
※給与明細の残業時間・残業代は前月分が反映されます。
例）5月給与（6/10支給）＝当月固定給＋4月勤怠実績
勤怠・給与給与明細の残業代が間違っている▼
まず給与明細の仕組み（MIKKEの給与明細説明資料）を確認の上、それでも誤りがある場合は #人事お困りごと相談「非公開問い合わせ」へご相談ください。
勤怠・給与AKASHIの締めができない▼
AKASHIお問い合わせフォームより詳細内容をご連絡ください。別途確認の上、回答いたします。
勤怠・給与AKASHI上で祝日が勤務日になっている▼
AKASHIお問い合わせフォームより詳細内容をご連絡ください。別途確認の上、回答いたします。
勤怠・給与36協定・勤怠管理・労働時間のルールについて知りたい▼
36協定とは、法定労働時間を超えて残業させる場合に労使間で結ぶ協定です（労働基準法第36条）。1日8時間・週40時間を超える場合に届出が必要です。
詳細はMIKKEの勤怠管理ページをご確認ください。
勤怠・給与住民税の納税通知書が届いたのですがどうすればいいですか▼
特別徴収（給与天引き）に切り替えるため、労務ML（G_ml_iico@sold-out.co.jp）へ納税通知書のPDFを提出してください。
※納付期限が過ぎているものは自身で納付の上、領収書コピーと未納分を提出。
勤怠・給与納会・総会の移動時間は業務時間扱いになるか▼
総会・納会時の会場への移動時間は業務時間扱いとなります。フルリモート適用の場合も交通費・宿泊費は経費精算可能です。
組織・異動カオナビの勤務地が間違っている▼
転勤・引越による場合はVPが「異動シートに記載して申請」。VPが申請済みなのに反映されていない場合は人事へ連絡ください。
組織・異動任用変更・職位変更・任用解除・職位解除を申請したい▼
新規任用・任用解除・任用変更すべて申請が必要です。VP以上から申請可能。【改定】職位・任用変更時の対応マニュアルをご参照ください。
組織・異動組織変更・任用・異動・権限委譲の申請スケジュールを知りたい▼
経営会議の日程に合わせて毎月変動します。イベント・〆切カレンダーをブックマークしてご活用ください（Googleカレンダーに追加可能）。
組織・異動自組織に人を採用したい場合の対応事項▼
VP以上から #人事お困りごと相談「非公開問い合わせ」へご相談ください。
組織・異動リーダーにAKASHIの権限委譲・評価の権限追加をしたい▼
直属のVPが #work_任用-組織-異動-お知らせなど_人事-vp以上 チャンネルで申請。チャンネル未参加の場合は #人事お困りごと相談「非公開問い合わせ」へ。
組織・異動AKASHI承認経路で前所属メンバーの申請が届く▼
正しい設定です。アルバイトの勤怠締め（毎月22日）が完了するまで、異動前の所属にも承認が届く仕様です。現上長は前所属メンバー分はご放念ください。
保険・証明書保険証はいつ届きますか▼
入社から2〜3週間でご自宅に届きます。届くまでに受診予定がある場合は「健康保険資格証明書」を各種証明書発行依頼フォームよりお申請ください。
保険・証明書保険証を紛失した場合の対応▼
MIKKEの再発行依頼フォーム（07_健康保険証を紛失したら）より申請。再発行まで2〜3週間。受診予定がある場合は「健康保険資格証明書」を発行依頼してください。
保険・証明書在籍証明書・就業証明書を発行してほしい▼
MIKKEの証明書発行フォームより依頼。原則押印省略（押印必須の場合はフォーム内で申請）。発行まで約2〜3週間。
対象：在籍証明書、就労証明書、勤務証明書、源泉徴収票（H30以前）、iDeCo証明書 など
保険・証明書限度額適用認定証について知りたい▼
被保険者本人がTJK健保組合HPより直接申請。20,000円以上の医療費に補助があります。会社経由での申請はできません。
保険・証明書源泉徴収票の提出は必要ですか▼
入社時に提出済みの場合は不要。年度内に他社からの収入がある場合は年末調整時に提出。未提出の場合は確定申告が必要です。
保険・証明書源泉徴収票を発行してほしい▼
毎年1月10日前後に発行。給与締め処理完了後、カオナビのプロファイルブックにアップロードされますのでご確認ください。
制度・福利厚生社員紹介・リファラル制度について知りたい▼
社員紹介制度があります。紹介方法・手当金額・対象など詳細はMIKKE（08_社員紹介制度について）をご確認ください。
制度・福利厚生資格取得報奨金（資格手当）について知りたい▼
カンパニー単位の資格取得手当は限定正社員も対象です。詳細はMIKKEの最新版福利厚生細則をご確認ください。
制度・福利厚生企業型確定拠出年金はありますか▼
ソウルドアウトグループには現在企業型確定拠出年金はありません。個人型（iDeCo）の対応はMIKKE（05_各種証明書）をご確認ください。
制度・福利厚生副業・兼業をする場合申請が必要ですか▼
雇用形態に関係なく副業申請が必要です。詳細はMIKKE（04_副業／兼業制度）をご確認ください。
制度・福利厚生限定正社員登用について教えてください▼
限定正社員登用制度の詳細はMIKKEの資料をご確認ください。
制度・福利厚生社内交際費・コミュニケーション費について知りたい▼
グランドルールはMIKKEをご確認ください。各カンパニーでの利用ルールは各カンパニーのSVPへお問い合わせください。
制度・福利厚生居住地・交通費100キロ圏内のルールを知りたい▼
居住地は自由です。フルリモート承認済みの場合は100キロ圏外でも交通費問題なし。リモート未承認かつ出張不可の場合、オフィスから100キロ超の交通費は自己負担。リモートワークはVP・SVPの許可が必要です。

#人事お困りごと相談 ワークフローの「非公開問い合わせ」へご相談ください。またはfirstcallの健康医療相談サービスをご活用ください。

クーポンコードを更新してください（最新コードはMIKKEで確認）。それでも解決しない場合は #人事お困りごと相談「公開問い合わせ」へ。
その他インフルエンザの予防接種・罹患時の対応▼
予防接種補助はMIKKEでご案内（「管理部門からのお知らせ&アナウンス」から「インフルエンザ」と検索）。罹患した場合、人事への報告義務・出社停止ルールはありません。上長と相談の上、就業判断してください。
その他退職・転職・離職の手続きについて知りたい▼
退職希望の場合は少なくとも30日前までに文書で退職願を提出。まず直属の上長に相談し、退職日等の調整後に上長経由でHRBPまたは人事へ連絡。その後、労務担当より手続き案内があります。
その他復職時にオリエンテーションはありますか▼
入社者オリエンテーション（1日・16日）にご参加いただきます。休業期間の長短問わず復職初日に参加（上長判断で変更の場合あり）。対象者は復職後Gカレにてご確認ください。
その他引っ越し後の住民票提出はPDFでなく写真でも可ですか▼
写真での提出でも問題ありません。ただし鮮明なものに限ります。確認が取れない場合は再提出をお願いすることがあります。
その他慶弔見舞金の申請について▼
リーダー以上が申請可能。配下メンバーのご親族に不幸があった場合、条件該当時に花輪/供花・弔電の手配が可能です。MIKKEのフォームから申請してください。
その他エンゲージメントサーベイ・GPTWで個人が特定されますか▼
経営・人事も個別回答は確認できません。5名未満の部署・セグメントは非表示のため、個人特定はできません。安心してご回答ください。

システムfreeeの支給月・勤怠月の見方がわからない▼
freeeは「支給月」ベースで管理しています。

例）5月表示 → 5/10支給分
　　4月基本給 × 3月勤怠 分となります。

※勤怠情報は1ヶ月ズレますのでご注意ください。
※支払日は毎月翌10日。10日が祝・休日の場合は前営業日。
システムfreeeで「末日締め（翌月10日払い）」と表示されているがアルバイトなので違う気がする▼
freeeの仕様上、全社員「末日締め（翌月10日払い）」と表示されますが、給与計算・支給には影響ありません。

実際の締め日・支払日：
・正社員・限定正社員・契約社員 → 末日締め（翌月10日払い）
・アルバイト → 15日締め（翌月10日払い）
